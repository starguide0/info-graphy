• Process (Defines): Git-flow, API 문서 기반 개발 등 새로운 개발 프로세스를 정의하고 도입한 경험을 근거로 팀의 효율성을 높이는 개발 프로세스를 새롭게 정의하고 도입합니다.

• Influence (Engineering): 특정 팀을 넘어 FE 기술 문화 전반에 긍정적인 영향을 주고, 타 팀의 문제 해결에 기여한 점을 고려하여 엔지니어링 전반에 긍정적 영향력을 발휘합니다.
